What features the Admin Gets ?

1. Add New Course/Language

Admins can add a new course or language by clicking the 'Add New Course' button that appears after logging in as an admin. They can provide the course name, chapter title, and upload the corresponding Markdown (.md) file.

2. Edit Course

Each course page provides an 'Edit Course' button. Admins can update the course by adding more chapters, editing existing chapter names, or uploading new .md files to replace existing ones.

3. Add New Chapter

Admins can add new chapters to an existing course by specifying the chapter title and uploading a new .md file. The chapter is immediately displayed in the course's chapter list.

4. Edit or Delete Chapter

Admins have the flexibility to edit an existing chapter's name, replace its .md file, or delete the chapter entirely from the course page.
