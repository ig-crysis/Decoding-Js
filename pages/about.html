About Decoding JS

Decoding JS is an educational platform designed to provide a user-friendly and intuitive interface for learning various languages concepts. Our goal is to simplify complex topics using well-structured notes in the form of Markdown (.md) files, making them easy to read and understand. We aim to create awareness about the potential of .md files for documentation and learning materials, while also offering a visually appealing and interactive learning experience.

Challenges Faced

  • Dynamic handling of Markdown files and rendering them seamlessly.
  • Ensuring secure authentication with JWT tokens and maintaining session security.
  • Managing dynamic folder creation and file uploads in a scalable manner.
  • Maintaining a consistent and responsive user interface.
  • Handling relative paths for seamless file fetching and display.
